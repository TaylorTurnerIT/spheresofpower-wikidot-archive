title:Martial Package
[[toc]]

+++ What is a Martial Package?
A Martial Package is a collection of martial talents, including a Martial Tradition, that can be quickly applied to NPCs or monsters to convert them into a Practitioner. Martial Packages are extremely similar to templates, but cannot be taken by player characters. (The player version is just getting martial talent progression in the first place.)

If a Martial Package grants proficiency in any weapon(s), armor(s), or shield(s), a creature that gains the Martial Package may also obtain any or all of the equipment it gains proficiency in.

++++ Building A Martial Package
A Martial Package consists of a Martial Tradition and 20 martial talents numbered and ordered. When a creature gains a Martial Package, it gets the talents up to its Base Attack Bonus. For example, a creature with a BAB of +7 gets the first seven talents in the martial package.

Each Martial Package should be thematic and designed to create a particular combat style. We particularly encourage basing ability progression on the basic Martial Tradition used to start it. For example, a Martial Package based on the Drunken Brawler tradition should probably focus on the Barroom and Wresteling spheres. Martial Packages can dip into spheres outside those it starts with, and when to do so is ultimately a matter of design judgement.

Martial Packages also provide a general description of the package and how it's intended to be used in play at higher levels.
